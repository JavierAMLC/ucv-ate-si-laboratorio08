## 11. Preguntas de reflexión

1. **¿Qué diferencia existe entre un chatbot y un agente?**
   Un chatbot convencional es reactivo y sigue un flujo conversacional rígido preprogramado. Un agente de IA posee un ciclo cognitivo basado en un modelo de lenguaje (LLM) que le permite razonar de manera autónoma, evaluar el contexto de la consulta y decidir dinámicamente qué herramientas externas ejecutar para resolver un problema.

2. **¿Qué función cumple Poetry?**
   Poetry es un gestor de dependencias moderno para Python que administra de forma transparente entornos virtuales aislados, garantizando la consistencia del proyecto mediante la fijación exacta de versiones a través del archivo `poetry.lock`.

3. **¿Por qué GitHub Actions es importante en DevOps?**
   Es fundamental porque automatiza los flujos de Integración Continua (CI). Permite que ante cada cambio subido al repositorio se ejecuten de forma automatizada tareas de compilación, ejecución de pruebas unitarias y análisis de calidad de software sin intervención manual.

4. **¿Qué problemas ayuda a detectar SonarQube?**
   Ayuda a detectar errores de lógica en el código (Bugs), vulnerabilidades o brechas de seguridad, malas prácticas de desarrollo (Code Smells) y mide de manera exacta el porcentaje de cobertura que tienen nuestras pruebas unitarias.

5. **¿Qué mejoras agregarías a este agente?**
   Implementar memoria persistente a largo plazo utilizando una base de datos para recordar interacciones previas del estudiante, e integrar una arquitectura RAG (Generación Aumentada por Recuperación) conectada directamente a los sílabos y reglamentos académicos de la universidad.
